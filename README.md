# [years-of-experience](https://years-of-experience.netlify.app) 📈📅

A simple web application for managing career profiles with offline capabilites.

## Frontend

---

## [years-of-experience](https://years-of-experience.netlify.app)

React/Typescript frontend for managing career profiles with experience history.

served via netlify: https://years-of-experience.netlify.app

#### Requirements

- Node 16
- git
- yarn (optionally)

#### Running Locally

```sh
git git@github.com:ajravinath/anuja-ranwalage-frontend-engineer-28June2022.git
cd anuja-ranwalage-frontend-engineer-28June2022/frontend
```

install dependencies and start application using

```sh
yarn
yarn start
```

or

```sh
npm install
npm run start
```

Your app should be running on [localhost:3000](http://localhost:3000/).

###### Testing

```sh
yarn test
npm run test
```

## Backend

---

## [years-of-experience](https://years-of-experience.herokuapp.com)

Node.js REST api using [Express 4](http://expressjs.com/) and postgres for managing career profiles with experience history.

hosted in heroku: https://years-of-experience.herokuapp.com

#### Requirements

- Node 16
- PostgreSQL 14
- git

#### Running Locally

```sh
git git@github.com:ajravinath/anuja-ranwalage-frontend-engineer-28June2022.git
cd anuja-ranwalage-frontend-engineer-28June2022/backend
```

copy `.env.example` file at `/backend` root to `.env` and replace environment variables with local setup.

install dependencies and start application using

```sh
npm install
npm run start
```

you can run a development server using

```sh
npm run dev
```

Your app should be running on [localhost:8000](http://localhost:8000/).
